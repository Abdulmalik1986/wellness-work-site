# Wellness Work Site

برنامج علاجي تفاعلي مبني على React و Supabase

## الوصف

جلسات الهولنس وورك - تطبيق ويب تفاعلي يرشد المستخدمين عبر مسارات علاجية متعددة للتأمل والتطور الشخصي.

## التكنولوجيا المستخدمة

- **React** - مكتبة JavaScript لبناء واجهات المستخدم
- **Supabase** - قاعدة بيانات PostgreSQL مفتوحة المصدر
- **Vercel** - منصة نشر حديثة

## البدء السريع

### المتطلبات
- Node.js و npm مثبتة على جهازك
- حساب على GitHub
- حساب على Vercel

### التثبيت المحلي

```bash
# استنسخ المستودع
git clone https://github.com/YOUR_USERNAME/wellness-work-site.git
cd wellness-work-site

# ثبت المكتبات
npm install

# شغّل المشروع محلياً
npm start
```

يفتح التطبيق تلقائياً على `http://localhost:3000`

## النشر على Vercel

1. ربط المستودع مع Vercel
2. أضف متغيرات البيئة (إن لزم الأمر)
3. اضغط Deploy
